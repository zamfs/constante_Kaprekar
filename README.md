# Constante de Kaprekar
A constante de Kaprekar pode ser calculada com qualquer número de 4 dígitos, que não possuam mais de 2 repetições. Após colocar os dígitos em ordem crescente e decrescente, subtrai o maior pelo menor. Após isso, repete-se o processo até chegar no número 6174, sendo que esse processo acontece em até 7 vezes.
